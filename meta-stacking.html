Meta-stacking

Combining prediction power of diverse machine learning models

Introduction:

Meta-stacking is an ensemble learning technoque that combines the predictions of multiple machine learning models to produce a final prediction which is more accurate than the individual predictions, usually.

Architecture:

Level 0 (base learners): Diverse models like SVM, Random Forests, CNNs, etc are trained on data. Each model makes a prediction.

Level 1 (Meta-learner): Predictions from base/level 0 are treated as features or input. A simple "meta-model" is trained on these new features to produce the final prediction.
